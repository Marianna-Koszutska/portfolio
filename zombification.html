Zombification

About the Project

It is a 2D puzzle game that involves solving maze-like levels. The player is a zombie virus that infects a healthy human's body by leaving a green trail behind it. There are 10 levels in total, each being a different body part. Can you solve all of them? Let's find out!

Game Features

  • Unique infection mechanic
  • Progressive difficulty system
  • Multiple organ systems to infect

My Responsibilities

Level Design

  • Designed 5 different infection scenarios representing different body systems
  • Created progressive difficulty curve matching immune system response

Coding

Narrative Design

  • Designed the tutorial sequence to teach mechanics organically
